# Captures d’écran

Ce dossier contient les images affichées dans le `README.md`.

## Remplacer les captures

Tu peux remplacer les fichiers SVG par tes captures (PNG/JPG/WebP) en gardant les mêmes noms de fichiers, ou en mettant à jour les liens dans `README.md`.

Fichiers utilisés :

- `popup-fix.png` : popup (onglet Fix)
- `popup-whitelist.png` : popup (whitelist / paramètres)
- `popup-export.png` : popup (onglet Export)
- `popup-migration.png` : popup (section Migration)
- `migration-panel.svg` : illustration du panneau de migration (SVG)
- `export.svg` : illustration export (SVG)
- `library-modal.svg` : illustration modal (SVG)
