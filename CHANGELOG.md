# Changelog

Ce projet suit le **Semantic Versioning (SemVer)** : `MAJOR.MINOR.PATCH`.

## [1.0.0] - 2026-01-01

### Added

- Correction automatique des images cassées (réécriture URLs `s00..s10` `/media/`).
- Popup avec toggles (Auto-fix, Debug) et gestion de whitelist.
- “Fix this page now” via injection one-shot.
- Export de la follow list (CSV/JSON) local, privacy-first.
- Panneau de migration assistée (`migrate.html`) avec recherches externes manuelles.
- i18n via `/_locales` (EN/FR).
- Tests Node simples pour utilitaires et export runner.
